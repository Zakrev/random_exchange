программа для проведения эксперементов по движку

main <-new|-con id> <-users n> <-days n>:
    -new        создать новую серию эксперементов (новый сайт, базу данных, дни считаются с 0)
                -users задает максимальное количество новых игроков, обязательная опция
    -con        продолжить эксперимент под номером id (папка в data рядом с программой)
                -users задает максимальное количество новых игроков, не обязательная опция
    -users      задает максимальное количество новых игроков (игроки будут создаваться не сразу, а имитируя приходящих)
    -days       количество эксперементальных дней
    
Структура файлов:
..
    main                    -   исполняемый файл
    data
        users_'id'           -   данные и конфигурации пользователей эксперимента №id
        data_'id'            -   данные и конфигурация эксперимента №id
            
Данные:
data
    информация
    -   id эксперимента
    -   текущая дата (дни от начала)
    -   кол-во пользователей
    -   кол-во денег (заработано на % от обменов)
    конфигурация
    -   интенсивность регистрации (чел/день)
    конфигурация обменов
    -   кол-во груп обменов
    конфигурация группы
    -   кол-во учавствующих
    -   кол-во победителей
    -   с нацекой/без наценки (1/0)
users
    информация
    -   id
    -   дата регистрации
    -   кол-во денег
    -   выйграно обменов
    -   проиграно обменов
    -   сыграно обменов
    -   текущая наценка
    -   всего получено
    -   всего потрачено
    конфигурация
    -   вероятность участия в обмене
    -   минимальный размер группы (исходный) для принятия участия
